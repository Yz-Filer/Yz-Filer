## 5. [OfficeShapeTool](https://github.com/Yz-Filer/OfficeShapeTool)  

![](image/hdl6.gif)  

Microsoft OfficeのPowerPoint形式（Office Open XMLのプレゼンテーション形式）で保存された図形に接続ポイントと調整ハンドルを追加するツールです。  
「図形の結合」機能とあわせて使うとオリジナルの図形が作成できます。
